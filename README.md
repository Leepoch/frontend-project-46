## System requirements
Node.js version: v17.9.0

Linux or Mac OS or WSL

## How to install

* Clone this repository
* Run the command: 
```
  make install
```
* Run the command (you may need `sudo`): 
```
  npm link
```

*****

## How to use

Just type in command line:

`gendiff --format <'stylish', 'plain' or 'json'> <path to file1> <path to file2>`
